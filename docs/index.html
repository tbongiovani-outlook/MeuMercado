🛒 Meu Mercado

Seu painel local para vender melhor no Mercado Livre.

Windows 10 / 11

Instalação em 1 clique

Baixe e execute — ele instala tudo sozinho e abre o site no final.

⬇️ Baixar instalador Depois de baixar, dê dois cliques em instalar.bat.

Como funciona

  1. Clique em Baixar instalador e salve o arquivo.
  2. Dê dois cliques em instalar.bat.
  3. Se o Windows avisar, clique em Mais informações → Executar assim mesmo.
  4. Aguarde: ele instala o Python (se preciso), baixa o app e prepara tudo.
  5. No fim, o site abre em http://127.0.0.1:8000 e um atalho é criado na área de trabalho.

Prefere pelo PowerShell?

Cole este comando no PowerShell (não precisa de administrador):

iex ((New-Object Net.WebClient).DownloadString('https://raw.githubusercontent.com/tbongiovani-outlook/MeuMercado/main/instalar.ps1'))

O que preciso ter?

  • Windows 10 ou 11. O instalador cuida do Python automaticamente.
  • Uma aplicação criada no DevCenter do Mercado Livre (Client ID e Secret). O próprio app te guia na configuração.
  • Nada é instalado para outros usuários nem exige senha de administrador.
